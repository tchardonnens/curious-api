## ğŸ“ Overview

The curious-api project is a Python backend service built on top of FastAPI that provides APIs for various functionalities such as chatbots, YouTube video search, and custom responses generated by AI models. It also includes endpoints for user authentication and CRUD operations on user data. The project uses popular libraries like Pydantic, SQLAlchemy, and Redis, and is built for easy deployment using Docker. It offers a valuable solution for building robust and scalable APIs with modern technologies in Python.
